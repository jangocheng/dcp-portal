
# 简介
采用前后端分离，前端UI:dcp-portal，后端微服务架构：dcp-platform基于Springcloud框架，搭建企业级微服务云框架，持续更新。 

基于ng-alain，进行第一次优化改造。

# 环境
- node 12.4
- angular 7.2.0

# 命令
```
$ git clone --depth=1 https://github.com/dunzung/dcp-portal.git my-project 
$ cd my-project && npm install && npm start
```

``
